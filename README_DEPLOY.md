# 舒尔特表格 - 可上线分享版

这是一个纯静态网页版本，直接部署即可，不需要后端。

## 文件说明

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`

## 推荐部署方式

### GitHub Pages

1. 新建一个 GitHub 仓库
2. 把当前文件夹里的所有文件上传到仓库根目录
3. 在仓库 `Settings -> Pages` 中开启 GitHub Pages
4. 选择从默认分支部署

### Netlify / Vercel / Cloudflare Pages

1. 新建站点
2. 上传当前文件夹内容，或者直接导入仓库
3. 发布目录留空或设置为根目录
4. 不需要构建命令

## 本地测试

双击 `index.html` 即可直接打开。

## 说明

- 历史最佳和设置保存在浏览器本地
- 不同用户之间不会同步数据
